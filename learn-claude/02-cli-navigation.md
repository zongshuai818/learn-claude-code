第 02 课：命令行导航
═══════════════════════════════════════════════════════════════════

了解几个命令行导航快捷键会让你效率更高。

## 键盘快捷键

| 快捷键         | 操作                               |
| -------------- | ---------------------------------- |
| `Enter`        | 发送消息                           |
| `Option+Enter` | 添加新行（用于多行输入）           |
| `Escape`       | 取消当前操作 / 清除输入            |
| `Ctrl+C`       | 中断 Claude，然后退出会话          |
| `Ctrl+L`       | 清屏                               |
| `Up/Down`      | 浏览消息历史                       |

## 斜杠命令

| 命令       | 功能                                   |
| ---------- | -------------------------------------- |
| `/help`    | 显示所有命令                           |
| `/clear`   | 清除对话历史                           |
| `/compact` | 总结对话以节省上下文空间               |
| `/config`  | 打开设置                               |
| `/cost`    | 显示 token 使用量和费用                |
| `/context` | 显示上下文窗口使用情况                 |
| `/model`   | 切换模型                               |
| `/quit`    | 退出 Claude Code                       |
