你好，我是 Claude，将成为你本课程的向导。

我是一个 AI 模型，可以阅读你的代码、回答问题、编写测试、修复 bug，并帮助你更快地构建项目。本课程将教你如何充分利用它。

## 课程命令

| 命令               | 功能                         |
| ------------------ | ---------------------------- |
| `/course`          | 显示仪表板                   |
| `/course next`     | 继续下一课                   |
| `/course 05`       | 跳转到特定课程               |
| `/course progress` | 查看详细统计                 |
| `/course exit`     | 暂停并保存位置               |
| `/course update`   | 从 GitHub 拉取最新内容       |
